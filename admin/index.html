🏷️ Categorías

▾ mostrar
Al renombrar una categoría, todos los productos que ya la usaban se actualizan automáticamente. Si eliminas una categoría que sigue en uso, esos productos conservan el nombre viejo pero ya no aparecerá en la lista para elegir en productos nuevos — te avisamos antes de borrar si hay productos afectados.

🏷️ Subcategorías

▾ mostrar
Las subcategorías son para uso interno: organizar mejor el Inventario y encontrar productos más rápido en Facturación (ej. buscar "Bebidas $2.500" muestra todas las bebidas de ese precio). No se muestran en la tienda pública. Al renombrar, los productos que ya la usaban se actualizan automáticamente.

Programa de puntos / fidelización

Solo aplica a clientes con cédula/NIT guardado en Facturación — un consumidor final ocasional no acumula puntos.
Ej: 1000 = el cliente gana 1 punto por cada $1.000 de su compra.

Cómo agregar un empleado: primero créale una cuenta en Firebase Console → Authentication → Users → Add user (correo + contraseña). Pídele que inicie sesión una vez en este panel (verá el panel vacío, es normal). Luego ve a Firebase Console → Authentication, copia su UID, y pégalo abajo.
